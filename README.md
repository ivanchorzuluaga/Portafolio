# Portafolio — Iván Zuluaga

Portafolio personal construido con Astro + Tailwind. Enfocado en presentar proyectos reales, experiencia y stack con una estética retrofuturista/post‑apocalíptica.

## Tecnologías

- Astro
- Tailwind CSS
- TypeScript
- Bun

## Desarrollo

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```

## Personalización rápida

- Datos y proyectos: `src/data/portfolio.ts`
- Secciones: `src/components/`
- Layout global: `src/layouts/Layout.astro`
- Tema y colores: `tailwind.config.mjs`

## Deploy

Vercel recomendado:

```bash
vercel
```
